### 🖥️ Prerequisites

Make sure you have the following installed:

* Python 3.8+
* Jupyter Notebook
* [Anaconda](https://www.anaconda.com/) (recommended)

You can install required packages using:

```bash
pip install notebook
```

Then, launch the notebook:

```bash
jupyter notebook LEC-1.ipynb
```
